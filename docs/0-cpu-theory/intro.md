# CPU 이론

[![bRd 3D](https://img.youtube.com/vi/Fg00LN30Ezg/mqdefault.jpg)](https://www.youtube.com/watch?v=Fg00LN30Ezg)

**bRd 3D** - CPU가 어떻게 작동하는지 시각적으로 설명

## 핵심요약

1. **트랜지스터 (스위치)**: 반도체 성질을 이용해 전기를 켜고 끄며 0과 1의 신호를 만듭니다.
2. **논리 회로 (계산기)**: 스위치들을 조합해 논리 게이트(AND, OR 등)를 만들고, 이를 통해 이진수 덧셈 같은 연산을 수행합니다.
3. **저장 장치**: CPU 내부에는 가장 빠른 메모리인 레지스터와 속도 차이를 줄여주는 캐시 메모리가 있어 데이터를 효율적으로 주고받습니다.
4. **작동 순서**: 메모리에서 명령을 가져오고(인출) → 분석해서(해석) → 계산하고(실행) → 저장하는 과정을 무한 반복합니다.
